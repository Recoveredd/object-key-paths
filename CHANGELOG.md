# Changelog

All notable changes to this package are documented here.

## 0.1.1 - 2026-05-12

- Added a `limit` option to stop traversal after a fixed number of returned path entries.

## 0.1.0 - 2026-05-12

- Initial release with key path, leaf path, segment and path entry helpers.
- Added dot and bracket path formatting.
- Added circular reference handling, max depth, array traversal and custom traversal support.
